# 圆桌 AI 平台 · 部署说明

## 准备工作（只需做一次）

### 第一步：注册 GitHub 账号
1. 打开 https://github.com
2. 点右上角 Sign Up，用邮箱注册
3. 记住你的用户名和密码

### 第二步：注册 Vercel 账号
1. 打开 https://vercel.com
2. 点 Sign Up → 选择 Continue with GitHub
3. 授权登录即可

---

## 部署步骤

### 第三步：上传项目到 GitHub
1. 登录 GitHub
2. 点右上角 "+" → New repository
3. Repository name 填：roundtable
4. 选 Public，点 Create repository
5. 点页面上的 "uploading an existing file"
6. 把 roundtable 文件夹里的所有文件拖进去
   （api 文件夹、public 文件夹、vercel.json）
7. 点 Commit changes

### 第四步：部署到 Vercel
1. 登录 Vercel
2. 点 Add New → Project
3. 找到你的 roundtable 仓库，点 Import
4. 直接点 Deploy（不用改任何设置）
5. 等待 1-2 分钟，看到 Congratulations 就部署成功了

### 第五步：填入 API Key
1. 在 Vercel 项目页面，点 Settings
2. 左侧菜单选 Environment Variables
3. 点 Add New
   - Name 填：OPENAI_API_KEY
   - Value 填：你的 API Key（sk-xxxxx）
4. 点 Save
5. 回到 Deployments，点最新那条右边的三个点 → Redeploy

### 第六步：访问你的网站
部署完成后，Vercel 会给你一个网址，类似：
https://roundtable-xxx.vercel.app

用浏览器打开，就可以使用了！

## 常见问题

**Q：API Key 从哪里弄？**
A：去 console.anthropic.com 注册，左侧 API Keys → Create Key
   需要充值至少 $5 才能使用（约 36 元）

**Q：每次讨论花多少钱？**
A：一次完整圆桌讨论（10条消息）大约消耗 $0.05-0.15，非常便宜

**Q：Vercel 要付费吗？**
A：个人使用完全免费，每月有足够的免费额度

**Q：网站网址能改吗？**
A：可以，在 Vercel → Settings → Domains 里可以绑定自己的域名
